# 矩阵

A: 2*3;   2行3列

B: 3*2;   3行2 列

相乘：A的行每一项*B的列每一项后相加 得出的结果 作为新矩阵坐标 [行 ] [列] 的结果

结果：2行2列的矩阵

```
A = [
 [2,3,-1],
 [6,1,-2]
];
B = [
 [4,-5],
 [-3,0],
 [1,2]
]

C = A * B;
A 第0行 和 B第0列相乘：   2*4 +3*-3 + -1*1 = -2      C[0][0] = -2
```
